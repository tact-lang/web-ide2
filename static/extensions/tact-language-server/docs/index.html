Key Features

Smart Code Completion

Intelligent code suggestions as you type. Get context-aware completions for variables, functions, and types. Smart snippets help you write common code patterns faster, while inline documentation keeps you informed.

Quick Navigation

Navigate through your codebase with precision. Jump instantly to definitions and find all usages across the project. Explore workspace symbols for quick access to any declaration, and use the file symbols tree to understand code structure at a glance.

Smart Documentation

Instant access to documentation for any symbol — types, functions, contracts, and more. Documentation is displayed when you hover over code elements.

Real-time Code Analysis

While you write, LS runs a set of checks and gives instant feedback if any errors are found, allowing you to quickly fix the code and continue development.

Signature Help

Get real-time information about function parameters as you type. See parameter types, names, and documentation without leaving your current context. Supports method calls, constructors, and native functions.

Smart Folding

Intelligent code folding that understands Tact's structure. Collapse contracts, functions, and code blocks to focus on what matters. Custom folding regions help organize large files.

Semantic Highlighting

Advanced syntax highlighting that understands code semantics. Different colors for variables, functions, types, and more. Distinguishes between mutable and immutable variables, highlights unused code.

Inline actionable information about your code. See references count, inheritance chain, implementation details. Quick access to related code and documentation.

Frequently Asked Questions

How to get started with Tact Language Server? +

Install the Tact Language Server extension in your preferred IDE (VSCode, WebStorm, etc.), open a Tact project, and you're ready to go. The server will automatically start providing smart completions and other features.

Which IDEs are supported? +

Tact Language Server works with any editor that supports the Language Server Protocol (LSP). This includes VS Code, Sublime Text, Atom, and many others.

How to report issues or request features? +

You can report issues and request features on our GitHub repository. For quick questions and community support, join Tact Telegram channel .

Is Tact Language Server free to use? +

Yes, Tact Language Server is completely free and open source. You can use it for both personal and commercial projects without any limitations.
